בשמחה גבי — ACL בלינוקס הוא אחד הנושאים שהכי משדרגים שליטה בהרשאות, במיוחד ב־CentOS. הנה הסבר ברור ומסודר שיעשה לך סדר.

---

# 🔐 מה זה ACL בלינוקס?
**ACL – Access Control Lists**  
זו מערכת הרשאות מתקדמת שמאפשרת לתת הרשאות *מדויקות* למשתמשים וקבוצות מעבר למודל הבסיסי של `owner/group/others`.

במילים פשוטות:  
ACL מאפשר לך לתת הרשאות *למשתמשים ספציפיים* או *קבוצות ספציפיות* על קובץ/תיקייה — גם אם הם לא הבעלים ולא חלק מהקבוצה הראשית של הקובץ.

---

# 🧩 למה בכלל צריך ACL?
המודל הרגיל של הרשאות (`rwx`) מוגבל:
- יש רק **משתמש אחד** בעלים
- יש רק **קבוצה אחת** בעלים
- כל השאר נכנסים ל־**others**

אם אתה רוצה לתת הרשאה למשתמש נוסף *בלי* לשנות בעלות — ACL פותר את זה.

---

# 🛠️ הפקודות המרכזיות של ACL

## 1) `getfacl` – הצגת ACL
מציג את כל ההרשאות המורחבות של קובץ/תיקייה.

```
getfacl file.txt
```

דוגמה לפלט:
```
# file: file.txt
# owner: root
# group: root
user::rw-
user:gabi:r--
group::r--
mask::r--
other::---
```

---

## 2) `setfacl` – הגדרת ACL

### 📌 לתת הרשאה למשתמש מסוים
```
setfacl -m u:gabi:rw file.txt
```
נותן למשתמש gabi הרשאות קריאה/כתיבה.

### 📌 לתת הרשאה לקבוצה מסוימת
```
setfacl -m g:developers:rwx file.txt
```

### 📌 להגדיר ACL רקורסיבי על תיקייה
```
setfacl -R -m u:gabi:rwx /data
```

### 📌 להגדיר ACL ברירת מחדל (default) על תיקייה
משפיע על קבצים חדשים שנוצרים בתוכה:

```
setfacl -m d:u:gabi:rwx /data
```

### 📌 למחוק הרשאה מסוימת
```
setfacl -x u:gabi file.txt
```

### 📌 למחוק את כל ה־ACL
```
setfacl -b file.txt
```

---

# 🧠 מושגים חשובים ב־ACL

## 🎭 ה־Mask
ה־mask מגדיר את ההרשאות המקסימליות ש־ACL יכול לתת.  
גם אם תיתן למשתמש `rwx`, אבל ה־mask הוא `r--`, הוא יקבל רק קריאה.

ניתן לשנות mask:
```
setfacl -m m:rwx file.txt
```

---

# 📁 דוגמה שימושית מהחיים
נניח שיש תיקייה `/var/www` ואתה רוצה לתת ל־apache ולמפתח מסוים גישה:

```
setfacl -m u:apache:rwx /var/www
setfacl -m u:gabi:rwx /var/www
```

בלי לשנות בעלות, בלי לשבור הרשאות קיימות.

# 🗂️ טבלה מסכמת

| פקודה | תפקיד |
|-------|--------|
| **getfacl** | מציג ACL של קובץ/תיקייה |
| **setfacl -m** | מוסיף/משנה הרשאות ACL |
| **setfacl -x** | מוחק הרשאה ספציפית |
| **setfacl -b** | מוחק את כל ה־ACL |
| **setfacl -R** | פעולה רקורסיבית |
| **setfacl -m d:** | הגדרת ACL ברירת מחדל לתיקייה |

---

- דוגמאות ל־ACL בסביבת Apache/Nginx  
- איך ACL עובד עם SELinux ב־CentOS  
- תרחישים נפוצים של DevOps/Storage  

תגיד מה מעניין אותך.
