
vi editor command

vi file_name - יוצר קובץ ונכנס אליו

Escap , i - מאפשר לכתוב לתוך הקובץ
Shift+ i - מאפשר לכתוב לתוך הקובץ

Esc , shift+ zz - שומר את המסמך ויוצא ממנו
Esc , :wq! - שומר את המסמך ויוצא ממנו

----------------------------------------------------------
יש לוודא שאנחנו על מצב רגיל ולא INSERT (יש ללחוץ על Esc)   
 

i              # מעבר למצב כתיבה בתחילת הסמן (Insert)
a              # מעבר למצב כתיבה אחרי הסמן (Append)
o              # פתיחת שורה חדשה מתחת ומעבר לכתיבה
O              # פתיחת שורה חדשה מעל ומעבר לכתיבה
I              # כתיבה בתחילת השורה הנוכחית
A              # כתיבה בסוף השורה הנוכחית
x              # מחיקת התו שמתחת לסמן
X              # מחיקת התו שלפני הסמן
dd             # מחיקת שורה שלמה
d$             # מחיקת כל מה שנותר מהסמן עד סוף השורה
d0             # מחיקת כל מה שלפני הסמן עד תחילת השורה
yy             # העתקת שורה שלמה (Yank)
y$             # העתקת טקסט מהסמן עד סוף השורה
y0             # העתקת טקסט מתחילת השורה עד הסמן
p              # הדבקה אחרי הסמן/שורה
P              # הדבקה לפני הסמן/שורה
u              # ביטול פעולה אחרונה (Undo)
Ctrl+r         # שחזור פעולה שבוטלה (Redo)
:w             # שמירת הקובץ
:q             # יציאה מהעורך
:wq            # שמירה ויציאה
:q!            # יציאה ללא שמירה (כפוי)
:w filename    # שמירה בשם חדש
:set number    # הצגת מספרי שורות
:set nonumber  # ביטול הצגת מספרי שורות
/text          # חיפוש טקסט קדימה בקובץ
?text          # חיפוש טקסט אחורה בקובץ
n              # מעבר לתוצאה הבאה בחיפוש
N              # מעבר לתוצאה הקודמת בחיפוש
gg             # קפיצה לתחילת הקובץ
G              # קפיצה לסוף הקובץ
0              # קפיצה לתחילת השורה
$              # קפיצה לסוף השורה
w              # קפיצה למילה הבאה
b              # קפיצה למילה הקודמת
:e filename    # פתיחת קובץ חדש בתוך VI
:r filename    # הכנסת תוכן של קובץ לתוך הקובץ הנוכחי
:42            # קפיצה לשורה מספר 42 (או כל מספר אחר)
